Resposive designe

Es una técnica de diseño y desarrollo web que mediante el uso de estructuras flexibles(contenedores flexibles, imágenes y video flexibles) y con CSS, logran adaptar un sitio web al entorno del dispositivo en el que se encuentre.

¿Que es HTML?

HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto

¿Que es CSS?

Es lo que se denomina lenguaje de hojas de estilo en cascada y se usa para estilizar elementos escritos en un lenguaje de marcado como HTML. CSS separa el contenido de la representación visual del sitio.

¿Qué es Javascript?

JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web, cada vez que una página web hace algo más que sentarse allí y mostrar información estática para que la veas, muestra oportunas actualizaciones de contenido, mapas interactivos, animación de Gráficos 2D/3D, desplazamiento de máquinas reproductoras de vídeo, etc.

¿Que es bootstrap?

Es un framework CSS desarrollado por Twitter para estandarizar las herramientas de la compañía. Sirve para desarrollar aplicaciones con diseño responsivo y estilizarla.

¿Qué es fundation?

Es un marco frontend responsivo que posee componentes HTML y CSS. Sirve para el diseño de interfaces de usuario, diversos fragmentos de código y plantillas.
